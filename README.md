# Minerals Museum

Museum with interactive 3D minerals.

[View website](https://mineralsmuseum.vercel.app)

### Technologies:
- TypeScript
- Next.js 14 (App Router)
- React Three Fiber
- Redux Toolkit
- Tailwind CSS
- Next UI
- Framer Motion (Animations)
- FSD (Feature Sliced Design)
- ESLint (Airbnb codestyle)
